# Nailspo — 사진으로 만드는 나만의 네일 디자인 (디자인 문서)

날짜: 2026-07-07
상태: 사용자 승인 대기

## 한 줄 요약

맘에 드는 사진(풍경, 무드보드, 옷 등)을 올리면, 그 사진의 색감·질감·분위기에 어울리는 네일 디자인 이미지를 AI가 생성해주는 모바일 웹앱. 생성된 이미지를 저장해서 네일샵에서 네일리스트에게 바로 보여주는 것이 최종 목적.

## 배경과 목적

- 네일아티스트들은 영감 사진(물, 꽃, 조개, 나비 등)을 보고 네일 디자인을 만든 뒤, 영감 사진과 디자인을 콜라주로 함께 올린다 (Desktop/Nailspo 참고 사진 3장).
- 이 "영감 사진 → 네일 디자인" 번역 작업을 손님이 직접 할 수 있게 한다.
- 손님은 네일 디자인을 검색하며 헤매지 않고, 자기 취향 사진 한 장으로 개인화된 시안을 얻는다.
- 사용자(개발자)의 목적: 한국 시장 대상 소규모 수익형 웹 유틸리티. 이번 단계는 수요 검증 MVP.

## 확정된 결정 사항

| 항목 | 결정 |
|------|------|
| 결과물 | AI 생성 네일 이미지 (실사 느낌) + 무드 키워드/색상 팔레트 텍스트 |
| 사용자 입력 | 사진 1장 + 손톱 모양(아몬드/라운드/스퀘어/스틸레토) + 길이(짧게/중간/길게) |
| 요금 | 무료, 1인당 하루 3회 제한 + 서비스 전체 하루 총량 제한 |
| 플랫폼 | 모바일 우선 웹앱 (설치 불필요, 링크로 접근) |
| 회원가입/DB | 없음 (MVP에서는 불필요) |
| 접근 방식 | 방법 A: Gemini 이미지 모델 1회 호출로 이미지+설명 동시 생성 |

## 화면 흐름 (3화면)

### 화면 1 — 시작
- 한 줄 설명: "맘에 드는 사진을 올리면, 나만의 네일 디자인이 나와요"
- 사진 업로드 버튼 (갤러리 선택 / 카메라 촬영)
- 업로드 후 옵션 노출:
  - 손톱 모양: 아몬드 / 라운드 / 스퀘어 / 스틸레토 (아이콘 선택)
  - 길이: 짧게 / 중간 / 길게
- "네일 디자인 만들기" 버튼

### 화면 2 — 생성 중
- 10~20초 대기 동안 단계별 문구 순환 표시 ("사진의 색감을 읽는 중...", "어울리는 파츠를 고르는 중...")

### 화면 3 — 결과
- AI 생성 네일 디자인 이미지 (손에 시술된 실사 느낌)
- 무드 키워드 2~3개 + 대표 색상 팔레트 (예: "청량한 아쿠아 · 글레이즈드")
- 버튼: 이미지 저장 / 다시 생성 / 새 사진으로 시작
- 남은 횟수 표시 (예: "오늘 2회 남음")
- "다시 생성"도 생성 1회로 차감된다 (매 생성마다 실제 비용이 발생하므로)

## 생성 파이프라인 (방법 A)

1. **클라이언트 리사이즈**: 업로드 사진을 브라우저에서 축소 후 전송 (속도/비용 절약).
2. **서버 API 라우트에서 Gemini 이미지 모델 호출** (API 키는 서버에만 보관):
   - 지시문: 사진의 색감·질감·분위기에서 영감받은 네일 디자인 / 선택한 모양·길이 반영 / 실제 손에 시술된 실사 사진 스타일 / 이미지와 함께 무드 키워드 2~3개 + 대표 색상 텍스트 반환.
3. **응답 표시**: 이미지 1장 + 설명 텍스트를 화면 3에 렌더링.

## 횟수 제한 (남용 방지)

- 서버 측에서 IP 기준 1일 3회 카운트, 자정(KST) 리셋. Upstash Redis(무료 플랜)에 기록.
- 브라우저 저장소만으로는 시크릿 창 우회가 가능하므로 서버 측 카운트가 필수.
- 서비스 전체 하루 총량 제한 200회(환경 변수로 조정 가능)를 추가로 두어 비용 폭주 방지.
- 생성 실패 시 횟수 차감하지 않음.

## 에러 처리

| 상황 | 처리 |
|------|------|
| AI 호출 실패/타임아웃 | "다시 시도해주세요" 안내, 횟수 미차감 |
| 부적절한 사진(AI 거부) | "다른 사진으로 시도해주세요" 안내, 횟수 미차감 |
| 개인 횟수 소진 | "내일 다시 만나요 🌙" 화면 |
| 전체 총량 소진 | "오늘 준비된 생성이 모두 끝났어요" 안내 |

## 기술 스택

- **프레임워크**: Next.js (프런트 + API 라우트 통합)
- **배포**: Vercel 무료 플랜
- **횟수 저장소**: Upstash Redis 무료 플랜
- **AI**: Google Gemini 이미지 생성 API (서버 측 호출만)
- **DB/인증**: 없음

## 검증 방법

- Desktop/Nailspo의 실제 네일리스트 콜라주 3장에서 영감 사진 부분을 입력으로 넣어, 원본 네일리스트의 디자인과 유사한 방향(색감·파츠·무드)의 결과가 나오는지 눈으로 비교 검증.
- 모바일 화면(iPhone 크기)에서 업로드 → 생성 → 저장 전체 흐름 동작 확인.
- 횟수 제한: 4회째 시도 시 차단되는지, 실패 시 차감 안 되는지 확인.

## 범위 제외 (이번 MVP에서 안 하는 것)

- 회원가입/로그인, 결제, 생성 이력 보관
- 네일샵 예약 연동, 네일리스트용 기능
- 다국어 (한국어만)
- 여러 장 사진 입력, 이미지 편집 기능
